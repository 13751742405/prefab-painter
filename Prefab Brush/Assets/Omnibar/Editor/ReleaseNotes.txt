v0.1.7 - 2019/01/23
===================
NOTE: On first launch the serialized state of the server will be removed

Added
-----
* Added Named Groups. https://trello.com/c/ktWhlvSZ/74-named-groups-for-gameobjects-selected
  You can save the current selection in the scene into a named group
  Save Group: just type SG=AwesomeGroup
  Remove Group: just type RG=AwesomeGroup
  Remove All Groups: just type RAG
  Filter: just type g: followed by whatever search string you want to search, e.g. g:ag to quickly get to AwesomeGroup


v0.1.6 - 2019/01/22
===================
NOTE: On first launch the serialized state of the server will be removed

Added
-----
* Support for Recent Items. They come up when launching the search bar (https://trello.com/c/ULE6wJTf/5-recent-items)
* Support for resetting pos/rot/scale for current selection (https://trello.com/c/GPVmmWBi/75-reset-pos-rot-scale-for-current-selection)
  Just do Reset Position, Reset Rotation, Reset Scale to reset the local position of the selected items.
  You can also use Reset World Position, Reset World Rotation, Reset Position Rotation Scale, Reset World Position Rotatation Scale.
  Of course you can just use the camel casing, e.g. RP to reset the position.
